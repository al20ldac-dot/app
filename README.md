## 🚀 Guía de Inicio Local (Visual Studio Code)

Sigue estos pasos para instalar y ejecutar el proyecto en tu computadora:

### 1. Requisitos Previos
Asegúrate de tener instalado:
- **Node.js** (v18 o superior): [Descargar aquí](https://nodejs.org/)
- **Git**: [Descargar aquí](https://git-scm.com/)
- **Visual Studio Code**: [Descargar aquí](https://code.visualstudio.com/)

### 2. Descarga y Preparación
1. Descarga el código desde Firebase Studio (Botón **Export** o **Download**).
2. Descomprime el archivo `.zip`.
3. Abre **Visual Studio Code**, ve a `Archivo > Abrir carpeta` y selecciona la carpeta descomprimida.

### 3. Instalación de Dependencias
Abre la terminal integrada en VS Code (Presiona `Ctrl + J`) y escribe:
```bash
npm install
```
*Este comando instalará automáticamente todas las librerías necesarias (React, Next.js, Firebase, Tailwind, etc.).*

### 4. Ejecución del Proyecto
Para iniciar el servidor de desarrollo:
```bash
npm run dev
```
Accede a [http://localhost:3000](http://localhost:3000) en tu navegador para ver tu simulador funcionando localmente.

## 📤 Cómo subir a GitHub

Sigue estos comandos en la terminal de VS Code para guardar tu proyecto en la nube:

1. **Inicializar Git**:
   ```bash
   git init
   ```

2. **Preparar archivos**:
   ```bash
   git add .
   ```

3. **Crear primer registro**:
   ```bash
   git commit -m "Versión estable Simulador 2026"
   ```

4. **Conectar con GitHub**:
   *Crea un repositorio en github.com y copia la URL (ej: https://github.com/tu-usuario/tu-repo.git)*
   ```bash
   git branch -M main
   git remote add origin REEMPLAZA_AQUÍ_CON_TU_URL_DE_GITHUB
   ```

5. **Subir archivos**:
   ```bash
   git push -u origin main
   ```
